xlsx_writer.py 전면 재작성 Distribution 차트 알고리즘 교체
Distribution 차트 헬퍼 시트 '_dist' 폐기 정리/정리_Y 방식으로 재작성

## Context

[xlsx_writer.py](client/report_generator/xlsx_writer.py) 의 distribution 차트가 1차 xlwings 변환 후 깨짐. 원인 : 기존 코드가 헬퍼 시트 '_dist' 1개에 LSL/USL/X/Y 를 2D grid 로 통합 기입 + 템플릿 1개 만들어 COM duplicate() 로 복제 + A1 문자열로 시리즈 참조하는 패턴. COM duplicate 후 series 마커 legend 리셋이 비결정적이고, 헬퍼시트 grid 좌표가 한칸만 어긋나도 차트 전체가 빈칸. 외부 변환에 취약

해결 :  검증된 다른 시스템에서 쓰던 안정적 패턴으로 교체
차트 데이터 시트를 **2개로 분리** (X값용, Y값용) 차트는 한개씩 독립 생성
USL/USL 은 시트 참조 없이 배열 리터럴 직접 지정.
컬럼 단위 정렬 + numpy broadcating 으로 벡터 처리
Dataframe 통째 1회 write (셀 단위 호출 금지)
차트 1개씩 독립 생성, Duplicate() 패턴 폐기 - 이부분은 변경하는게 이득일지 유지하는게 이득일지 너가 검토해
LSL_USL 은 시트 참조 없이 COM 배열 리터럴 - 이부분도 변경하는게 이득일지 유지하는게 이득일지 너가 검토해 

전체 흐름은 xw.App 컨텍스트 1개 안에서 단일 세션. write() 시그니처 변경 X Diff (a_only/b_only) / PNG 부착은 기존 코드 유지

1-2. ECDF (numpy broadcasting, 1초과는 NAN)
def sort_data_to percent(df) :
inds = np.arange(df.shape[0]).reshape(-1,1) + 1 #(N,1)
counts = df.notna().sum().to_numpy() #(n_cols,)
vals = np.divide(inds,counts, out=np.zeros(df.shape,dtype=float), where counts!=0)
vals[vals > 1] = np.nan
return pd.Dataframe(vals, colums=df.columns)

1-3 cource리스트 -> X/Y dataframe 리스트
def get_list_xvalues_by_each_list(source_dfs):
return [sort_alldata(df,True).reset_index(drop=True) for df in source_dfs]

def get_list_yvalues_by_dut_from_xvalues(x_dfs):
return [sort_data_to_percent(df).reset_index(drop=true) for df in xdfs]
