Honey ZIP release guide
=======================

Honey 배포는 PyInstaller onedir 결과물(client/dist/Honey/)을 ZIP으로 묶어
server/releases/에 올리는 방식이다. 별도 설치 프로그램은 사용하지 않는다.

자동 배포
---------

client/release/에서 실행:

    .\release_honey.ps1 -Version 3.0.1 -Notes "변경 사항 요약"

스크립트 작업:

1. client/transport/config.py CURRENT_VERSION 갱신
2. PyInstaller onedir 빌드
3. client/release_dist/Honey-<version>.zip 생성
4. server/releases/Honey-<version>.zip 복사
5. server/releases/version.json 갱신
6. server/releases/release_log.txt 기록

더블클릭 빌드
-------------

    client/build_zip.bat

필수 준비물
-----------

- Python 3.10+
- client/requirements.txt 설치
- PyInstaller

설치 예:

    cd F:\COINAPI\report_server\client
    python -m pip install -r requirements.txt
    python -m pip install pyinstaller

업데이트 동작
-------------

1. 클라이언트가 /honey/version 조회
2. 새 version이면 /honey/download에서 ZIP 다운로드
3. sha256 검증
4. frozen exe인 경우 ZIP을 임시 폴더에 압축 해제
5. 외부 bat가 현재 Honey 종료를 기다림
6. 새 파일을 앱 폴더에 복사하고 Honey.exe 재실행

수동 롤백
---------

server/releases/version.json이 가리키는 file/version/sha256만 이전 ZIP으로 되돌리면 된다.
